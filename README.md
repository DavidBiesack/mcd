# Example Commander.js CLI app

Read the blog post: [Deeply nested subcommands in Node CLIs with Commander.js](https://maximilianschmitt.me/posts/nested-subcommands-commander-node-js/)

## Commands

```
Program overview:
  $ mcd

Order some food:
  $ mcd order

Order a burger menu:
  $ mcd order burger

Order some dessert:
  $ mcd order dessert

Pay a meal:
  $ mcd pay
```
